Getting Started

  1. Enter the MTLBooks novel URL you want to crawl
  2. Choose your preferred settings (or use defaults)
  3. Click "Start Crawl" to begin
  4. Monitor progress in the Active Crawls section

Chapter Range Examples

  • All Chapters: Downloads the entire novel
  • 1-50: Downloads chapters 1 through 50
  • Custom: Use formats like "1-10,25,50-100"

Performance Tips

  • Start with concurrency 2-3 for stability
  • Use delays of 1000-2000ms to be respectful
  • Headless mode is faster but visible mode helps with debugging

Output Formats

  • Markdown: Always generated (individual chapter files)
  • EPUB: E-book format for reading devices
  • TXT: Plain text format for simple reading
